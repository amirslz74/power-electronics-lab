# power-electronics-lab

Advanced power electronics, MOSFET driver circuits, switching converters, and embedded control systems.

This repository contains a collection of real-world power electronics projects, including:

- Buck converters
- Boost converters
- Half-bridge MOSFET drivers
- Synchronous switching designs
- STM32-controlled power stages
- EMI/EMC optimized PCB layouts
- High-current switching systems
- Analog feedback and sensing circuits

---

# Features

- Production-oriented PCB designs
- Thermal-aware layouts
- EMI/EMC considerations
- Oscilloscope waveform validation
- Real hardware prototypes
- Embedded firmware integration
- STM32 / ESP32 compatible designs
