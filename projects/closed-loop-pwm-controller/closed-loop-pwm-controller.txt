# Current-Controlled PWM Power Stage

STM32-based PWM power control system with closed-loop current regulation for high-current switching applications.

## Overview

This project focuses on the development of a high-frequency PWM power stage with real-time current control and feedback regulation.

The system is designed for applications requiring:

- Accurate current limiting
- Stable PWM-driven power delivery
- Fast transient response
- MOSFET-based high-current switching
- Analog current sensing
- Embedded closed-loop control

The controller is based on STM32 and supports configurable PWM frequency, duty-cycle management, and current feedback monitoring.
